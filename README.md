# Java Tech Store (MongoDB)

A simple Tech Store management application built in Java with MongoDB persistence.

##  Features
- Add and manage tech items:
  - Phones
  - Laptops
  - Televisions
- Object-oriented item hierarchy
- MongoDB integration for storing items
- Basic CRUD logic

##  Tech Stack
- Java
- MongoDB
- JSON Deserialization

##  Run

1. Install MongoDB and start it locally (or use an online cluster).
2. Update the MongoDB connection string in the code if needed.
3. Compile and run:

```bash
javac *.java
java Main
